# Управление пользователями

Современный фронтенд-интерфейс для панели администратора, разработанный на **React 18**, **TypeScript** и **Vite**.
Проект включает в себя полноценный CRUD-функционал для управления списком пользователей, серверную пагинацию, детальные
страницы и валидируемые формы, а также полное покрытие ключевой логики юнит- и интеграционными тестами.

---

## 🚀 Стек технологий

* **Core:** React 18, TypeScript, Vite
* **State Management:** Redux Toolkit (@reduxjs/toolkit, react-redux)
* **UI Components:** Material UI (MUI v5), MUI Icons
* **Routing:** React Router DOM v6 (Data Routers / Layout паттерн)
* **HTTP Client:** Axios
* **Testing Suite:** Vitest
* **Backend Simulation:** MockAPI.io
